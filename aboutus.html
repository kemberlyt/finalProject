About Girls Who Code



Girls Who Code was founded by Reshma Saujani with the purpose of eliminating the gender gap in technology. The Girls Who Code Summer Immersion Program is a 7-week, immersive experience in which participating students learn Computer Science through creative projects, gain exposure to careers in the tech industry, and join a sister hood of girls who code across the country.
